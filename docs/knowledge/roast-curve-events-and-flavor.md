# 預計曲線事件與階段如何影響風味

這篇是在解釋烘焙計畫裡的「預計曲線」要怎麼看。

預計曲線不是要你把每一秒都烘得跟表格一樣，而是先設定幾個重要檢查點。烘完之後，你可以用這些檢查點回頭判斷：這鍋的酸、甜、花香、水果調、草青味或烘焙味，可能是在哪個階段被拉開差距的。

以下用目前這鍋 `100g` 衣索比亞南西寶水洗藝伎的目標來說明：

| 事件 | 目標時間 | 主要觀察 |
| --- | --- | --- |
| 入豆 | `0:00` | 機器熱狀態是否穩定 |
| 回溫點 | `0:40-0:55` | 前段熱量是否足夠 |
| 乾燥結束 | `4:05-4:35` | 青草味是否退掉，豆子是否轉黃 |
| 一爆開始 | `7:55-8:30` | 熱是否進到豆芯，是否有足夠動能進入發展 |
| 下豆 | `9:05-9:35` | 酸甜是否平衡，花果是否保留 |

## 事件和階段差在哪裡

事件是烘焙過程中的「時間點」。例如入豆、回溫點、乾燥結束、一爆開始、下豆。

階段是兩個事件之間的「一段過程」。例如：

- 入豆到乾燥結束：乾燥期。
- 乾燥結束到一爆開始：梅納反應期。
- 一爆開始到下豆：一爆後發展期。

事件幫你定位問題。階段幫你理解風味是怎麼被建立或破壞的。

## 入豆：先確認機器熱狀態

入豆是生豆倒進烘豆機的時間點，英文常叫 `charge`。

這一步看起來只是開始，但它會影響整鍋的節奏。如果機器預熱不穩，可能會出現兩種問題：

| 狀況 | 可能風味 |
| --- | --- |
| 機器太冷就入豆 | 前段推不動，後面容易拖長，喝起來悶、空、甜感不清楚 |
| 機器太熱就入豆 | 前段衝太快，豆表面容易受熱過猛，喝起來可能粗糙、酸尖或有焦感 |

所以計畫裡寫「若機器預熱不穩，先不要入豆」，不是形式上的提醒。對初學者來說，穩定預熱比追求漂亮曲線更重要。

這鍋目前建議：

- 入豆量：`100g`
- 入豆 BT：約 `155-158°C`
- SV：預熱 `155-158°C`，入豆後調到 `215°C`
- Drum：`90%`
- Air：`60%`

## 回溫點：看前段熱量有沒有接上

回溫點是入豆後豆溫先下降，降到最低後重新開始上升的地方，英文是 `turning point`。

剛入豆時，生豆是室溫，會把烘焙室熱量吸走，所以 BT 下降是正常的。重點不是它有沒有下降，而是它什麼時候回頭往上。

這鍋目標回溫點是 `0:40-0:55`。

| 偏離狀況 | 可能原因 | 可能風味 |
| --- | --- | --- |
| 太早回溫 | 入豆溫或前段熱量偏高 | 豆表面受熱快，可能酸質銳利、口感粗 |
| 太晚回溫 | 入豆溫偏低、火力不足、風量帶走太多熱 | 後面容易拖，可能悶、甜感弱、風味不立體 |

回溫點不直接決定好不好喝，但它是第一個提醒你「前段熱量是否合理」的訊號。

## 乾燥期：把青草水氣處理乾淨

乾燥期是入豆到乾燥結束這一段。這段的任務是把生豆內外熱起來，讓生豆味、青草味和一部分水氣慢慢退掉。

這裡的「乾燥」不要理解成把豆子烘乾就好。它其實是在幫後面的甜感和香氣打基礎。如果前段沒處理好，後面就算一爆時間看起來正常，喝起來也可能怪。

這鍋目標乾燥結束是 `4:05-4:35`。

| 乾燥期狀況 | 可能風味 |
| --- | --- |
| 太短，早於 `3:50` | 前段熱太猛，可能花香被壓掉，酸質變尖，豆面甚至有焦點 |
| 太長，晚於 `4:50` | 熱進得慢，可能青草味、穀物感、悶味比較明顯 |
| 接近目標 | 青草味退掉，後面比較容易建立乾淨甜感 |

初學者可以觀察三件事：

- 豆色從綠轉黃。
- 氣味從青草、豆腥，轉成穀物、麵包、蒸玉米感。
- RoR 不要在這時候已經掉得很虛。

## 梅納反應期：建立甜感和水果輪廓

梅納反應期是乾燥結束到一爆開始這一段。梅納反應（Maillard reaction）可以先理解成：豆子開始從單純的黃豆、穀物味，往甜感、香氣、顏色和口感發展。

這段會影響你想要的桃子、紅蘋果、棉花糖甜感能不能出來。

這鍋目標是：

- 乾燥結束：`4:05-4:35`
- 一爆開始：`7:55-8:30`
- 梅納反應期大約 `3:40-3:55`

| 梅納反應期狀況 | 可能風味 |
| --- | --- |
| 太短 | 酸質明亮但可能薄，甜感不夠，像檸檬水但沒有果肉感 |
| 太長 | 酸會變低，甜感可能變厚，但花香、水果調容易變鈍 |
| 熱量不足 | 可能有空洞、木質、穀物感 |
| 熱量穩定下降 | 較容易出現乾淨的蘋果、桃子、白糖水感 |

水洗藝伎通常希望乾淨、明亮、花果清楚。這代表梅納反應期不能太短，也不能拖到把花香磨掉。第一鍋先抓中間值，烘完再用杯測決定下一鍋要延長還是縮短。

## 一爆開始：確認熱有沒有進到豆芯

一爆是豆子內部水蒸氣壓力讓結構裂開，聲音像比較低沉的爆裂聲，英文是 `first crack` 或 `FC`。

一爆開始不只是聲音事件，它代表豆子內部已經累積到某個程度的熱與壓力。對淺焙來說，一爆品質很重要。

這鍋目標一爆開始是 `7:55-8:30`。

| 一爆狀況 | 可能風味 |
| --- | --- |
| 早於 `7:40` | 整體速度偏快，可能酸質銳利、香氣漂亮但甜感不足 |
| 晚於 `8:40` | 中段推進偏慢，可能悶、甜感不清楚、花果不夠亮 |
| 一爆很弱、零星 | 熱可能沒有均勻進到豆芯，可能喝起來空、草青或酸尖 |
| 一爆穩定、連續 | 比較有機會酸甜平衡，風味完整 |

標記 `FC START` 時，不要聽到一兩聲就按。等到開始連續爆裂再標，這樣後面計算發展時間才有意義。

## 一爆後發展期：把酸轉甜，但不要烘掉花果

一爆開始到下豆這段，叫一爆後發展期，也常簡稱發展期（development）。

這段會大幅影響酸甜平衡。對你這支水洗藝伎來說，目標不是把酸壓掉，而是讓檸檬、蘋果、桃子的酸變得比較圓，後面有棉花糖或白糖水的甜感。

這鍋目標：

- 一爆後發展：`1:05-1:20`
- 下豆總時間：`9:05-9:35`
- 下豆 BT：約 `192-194°C`

| 發展期狀況 | 可能風味 |
| --- | --- |
| 太短 | 酸尖、草青、甜感少，可能像水果還沒熟 |
| 太長 | 花香下降，桃子和蘋果變鈍，可能出現烘焙味 |
| RoR crash | 悶、空、甜感斷掉 |
| RoR flick | 粗糙、苦感、尾韻不乾淨 |

第一鍋抓 `1:10` 左右很合理。它不是極限淺焙，但比較能降低初學者常見的「香是香，但喝起來酸薄空」的風險。

## 下豆：決定這鍋最後的風味落點

下豆是把豆子倒出烘豆機並開始冷卻的時間點，英文是 `drop`。

下豆不是只看溫度。尤其 Kaleido M1 搭配 Artisan 時，BT 數值會受到探針位置、批次量、風量、設定方式影響。你要一起看：

- 一爆後過了多久。
- BT 增加了幾度。
- 一爆聲是否還在密集進行。
- 豆色是否均勻。
- 氣味是甜香、花果，還是開始有煙味、烘焙味。

對這鍋來說，如果一爆在 `188-189°C` 左右開始，優先考慮 `192-193°C` 附近下豆。如果一爆比較晚、BT 比較高，才看 `193-194°C`。

## 用風味回頭修曲線

烘完後不要只問「曲線漂不漂亮」，要用喝到的結果回頭看事件。

| 喝起來 | 優先回頭看 |
| --- | --- |
| 草青、穀物、生味 | 乾燥期是否太短或一爆是否太弱 |
| 酸尖、甜感少 | 發展期是否太短，或梅納反應期是否不夠 |
| 悶、空、香氣不開 | 乾燥期或梅納反應期是否拖太久，RoR 是否 crash |
| 苦、煙、尾韻粗 | 一爆後是否太熱，RoR 是否 flick，排煙是否不足 |
| 花香、水果清楚但太薄 | 可以下鍋多發展 `5-10` 秒 |
| 酸甜平衡、桃子蘋果清楚 | 先不要大改，只做很小的調整 |

## 跟 Kaleido M1 / Artisan 的實作關聯

在 Artisan 裡，最重要的是把事件按準：

- `CHARGE`：生豆真的進機器。
- `DRY END`：豆色轉黃、青草味退掉。
- `FC START`：連續一爆開始。
- `DROP`：豆子真的離開烘焙室。

Kaleido M1 是小型電熱滾筒機，`100g` 批次反應會比 `150g` 更敏感。這代表火力、風量、SV 的調整都不要一次改太多。第一鍋先把事件記準，比追求完美曲線更重要。

如果這鍋是第一次烘，最實用的目標是：

- 乾燥結束落在 `4:05-4:35`。
- 一爆開始落在 `7:55-8:30`。
- 一爆後發展約 `1:10`。
- 烘完保留 Artisan 曲線，杯測後再決定下一鍋怎麼改。
