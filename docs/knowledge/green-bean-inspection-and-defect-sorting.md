# 生豆烘焙前怎麼檢查與挑瑕疵

這篇是給烘豆前 10 分鐘看的。目標不是把生豆品管做到專業出口檢驗等級，而是讓初學者知道：哪些東西一定要挑掉，哪些狀況要記錄，哪些不要太緊張。

以你目前這支衣索比亞南西寶水洗藝伎來說，標示是 G1，理論上瑕疵會比較少。但 G1 不代表完全沒有瑕疵。尤其你現在用 `100g` 小批次烘，一顆壞豆、一顆石頭、幾顆明顯發霉或蟲蛀豆，都會比大批次更容易影響結果。

## 為什麼烘豆前要檢查生豆

檢查生豆主要有三個目的：

1. 避免異物傷機器。
2. 降低明顯瑕疵豆造成的怪味。
3. 讓每一鍋的條件更一致，方便回顧。

初學者很容易只看烘焙曲線，但如果生豆裡有明顯瑕疵，烘焙再漂亮，杯子裡還是可能有霉味、土味、藥水味、尖酸、發酵臭或髒感。

## 烘豆前準備

建議準備：

- 白色盤子或淺色托盤。
- 夾子或手挑用的小碗。
- 小秤。
- 乾淨的密封袋或罐子。
- 手機拍照，方便留下紀錄。

環境要夠亮。不要在黃光很重的地方挑豆，因為顏色會失真。

這支豆子如果計畫入豆 `100g`，建議先秤 `102-103g` 來檢查。挑掉瑕疵後，再補到實際入豆 `100g`。這樣計畫裡的入豆量才不會被挑豆動作影響。

## 第一步：先看有沒有異物

先找這些東西：

| 類型 | 怎麼看 | 怎麼處理 |
| --- | --- | --- |
| 石頭 | 顏色灰、白、黑，形狀不像豆子，手感硬 | 一定挑掉 |
| 木屑、枝條 | 棕色纖維狀、形狀不規則 | 一定挑掉 |
| 金屬、塑膠、線頭 | 明顯不是咖啡豆 | 一定挑掉 |

異物不是風味問題而已，可能傷到磨豆機，也可能在烘豆機裡造成不必要的風險。

## 第二步：聞生豆氣味

把生豆放在手上或杯子裡，靠近聞一下。

正常生豆可能有：

- 青豆、青草、豆莢味。
- 穀物味。
- 淡淡花香或甜味。
- 水洗豆常見的乾淨、清爽感。

需要注意的味道：

| 氣味 | 可能問題 | 怎麼處理 |
| --- | --- | --- |
| 霉味、濕紙箱味 | 可能保存受潮或發霉 | 不建議烘，至少先隔離確認 |
| 土味、藥水味、腐敗味 | 可能有嚴重瑕疵或污染 | 不建議直接混進這鍋 |
| 很重的發酵臭 | 可能處理或保存異常 | 先少量試烘，並清楚記錄 |

如果整包豆子聞起來明顯不對，不要只挑幾顆就硬烘。那通常不是單顆瑕疵能解決的問題。

## 第三步：挑掉一定要移除的瑕疵

以下這些建議直接挑掉。

| 瑕疵 | 外觀 | 可能帶來的風味 |
| --- | --- | --- |
| 黑豆 | 整顆或大部分變黑、深褐 | 霉味、腐敗味、髒感 |
| 發霉豆 | 表面有白、灰、綠色霉斑，或看起來毛毛的 | 霉味、濕紙箱味，不建議入口 |
| 嚴重蟲蛀豆 | 有明顯孔洞、內部被蛀空 | 木質、髒感、酸敗感 |
| 嚴重破碎豆 | 破到剩半顆以下，邊緣很碎 | 烘焙時容易過度受熱，帶苦焦 |
| 貝殼豆嚴重破片 | 很薄、空心、像豆殼碎片 | 容易烘焦，讓杯中有雜味 |
| 異色豆 | 顏色明顯跟其他豆不同，偏黑、偏褐、怪黃 | 可能帶出不一致風味 |

如果你不確定某一顆要不要挑，可以先放到「可疑豆」小碗。最後拍照，再決定要不要放回去。第一鍋建議保守一點，明顯怪的先挑掉。

## 第四步：哪些不用太緊張

不是每一顆看起來不完美的豆子都要挑掉。

| 狀況 | 通常怎麼看 |
| --- | --- |
| 大小有些差異 | 衣索比亞豆常見，不一定是瑕疵 |
| 顏色深淺略不同 | 水洗豆也可能有自然差異 |
| 很小的缺角 | 少量可以接受 |
| 銀皮殘留 | 正常，烘焙中會脫落 |
| 豆面皺摺 | 高海拔豆常見，不代表壞掉 |

初學者挑豆容易走向兩個極端：完全不挑，或每顆都想挑到完美。比較務實的做法是，只挑掉「明顯會造成問題」的瑕疵。

## 第五步：特別注意未熟豆和 Quaker

未熟豆是成熟度不足的豆子。烘完後如果顏色特別淡、像米黃色或花生色，常被叫做 `quaker`。Quaker 在熟豆裡比較容易看出來，生豆階段不一定好辨認。

生豆階段可以注意：

- 顏色特別淡、偏黃白。
- 顆粒比較小、比較輕。
- 表面看起來不飽滿、皺縮。
- 跟同批豆子的顏色差很多。

它可能帶來的風味：

- 花生皮。
- 穀物味。
- 紙味。
- 甜感不足。
- 酸質粗糙。

但不要為了追 quaker 把所有顏色略淡的豆子都挑掉。對初學者來說，先挑掉明顯異常的就好。烘完後再看熟豆，如果有特別淡、特別不熟的豆子，可以拍照記錄，下次再回頭對照生豆外觀。

## 第六步：挑完後重新秤重

如果計畫入豆 `100g`，流程建議是：

1. 先秤 `102-103g` 生豆。
2. 平鋪檢查。
3. 挑掉異物與明顯瑕疵。
4. 重新秤重。
5. 補足或扣到 `100g`。
6. 才開始預熱與入豆。

這樣做的原因是：烘焙計畫裡的火力、風量、時間都是以 `100g` 為基準。如果你挑完只剩 `96g`，但仍照 `100g` profile 烘，曲線可能會偏快，結果也比較難比較。

## 要記錄什麼

不用寫很複雜，先記這些就好：

| 項目 | 範例 |
| --- | --- |
| 檢查前重量 | `103.0g` |
| 挑掉重量 | `1.2g` |
| 最後入豆重量 | `100.0g` |
| 挑掉內容 | 破碎豆 4 顆、疑似蟲蛀 2 顆、黑褐色豆 1 顆 |
| 生豆氣味 | 乾淨、青草、淡花香，無霉味 |
| 照片 | 生豆整體、挑掉瑕疵豆 |

可以把這段記在烘焙計畫的「生豆資料」或「實際紀錄」下面。

## 這支豆子的建議處理方式

以目前這支 `317 跳舞山羊 - 西阿希 南西寶 G1 - 西達摩 - 藝伎 - 水洗處理法` 來看，第一鍋可以這樣做：

1. 從袋中取 `102-103g`。
2. 平鋪在白盤上。
3. 先聞整體氣味，確認沒有霉味、濕紙箱、腐敗味。
4. 挑掉石頭、木屑、黑豆、發霉豆、嚴重蟲蛀豆、嚴重破碎豆。
5. 對可疑未熟豆先保守挑掉明顯異常的，不要過度挑。
6. 重新秤到 `100g`。
7. 拍一張挑掉瑕疵的照片。
8. 再開始預熱 Kaleido M1。

## 烘完後也要看熟豆

挑生豆只是第一步。烘完後可以再檢查熟豆：

- 有沒有顏色特別淡的 quaker。
- 有沒有表面焦點。
- 有沒有破碎豆變得很深。
- 豆色是否整體均勻。

如果熟豆裡有明顯 quaker，可以挑掉幾顆再沖煮。然後記錄數量。這會幫你判斷：杯中的穀物味或紙味，是烘焙發展不足，還是生豆成熟度造成的。

## 常見誤解

| 誤解 | 比較務實的看法 |
| --- | --- |
| G1 就不用挑 | G1 瑕疵少，但小批次還是建議看一遍 |
| 每顆顏色不同都要挑掉 | 只挑明顯異常，否則會過度挑豆 |
| 生豆洗一洗比較乾淨 | 不建議隨便水洗生豆，會改變含水狀態，也增加烘焙不確定性 |
| 瑕疵都能靠烘焙修掉 | 嚴重霉味、污染、腐敗味通常不是烘焙能救的 |
| 熟豆有淺色豆一定是烘壞 | 可能是 quaker，也可能是豆子本身成熟度問題，要搭配生豆和杯測看 |

## 跟 Kaleido M1 / Artisan 的關聯

Kaleido M1 的 `100g` 批次很小，生豆狀態的差異會被放大。挑掉幾顆瑕疵豆、固定最後入豆重量、記錄挑豆結果，會讓 Artisan 曲線和杯測結果更容易比較。

如果每次入豆重量都差很多，或有時挑豆有時不挑，下一次要判斷火力、風量、SV、Drum 的調整就會變得很困難。

第一階段先做到這三件事：

- 每次都檢查異物與明顯瑕疵。
- 每次都固定最後入豆重量。
- 每次都拍照或簡單記錄挑掉什麼。
