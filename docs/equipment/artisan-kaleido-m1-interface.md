# Artisan + Kaleido M1 介面按鈕入門

這篇是給第一次使用 Kaleido M1 搭配 Artisan 烘豆時看的操作地圖。目標不是一次學會所有功能，而是先知道一鍋烘焙過程中哪些按鈕一定會用到、每個按鈕代表什麼。

目前依據：

- 使用機器：Kaleido M1，200g 等級批次。
- 烘焙軟體：Artisan。
- 介面參考：使用者目前的 Artisan 截圖。

如果畫面左上顯示 `Kaleido disconnected`，代表 Artisan 尚未連上 Kaleido。正式烘豆前，應該先確認連線成功，且右側溫度讀值有正常變化。

## 一鍋烘焙時，大概會按哪些按鈕

先記這個順序就好：

1. 按 `CONTROL`：讓 Artisan 進入控制 Kaleido 的狀態。
2. 按 `START HEATING`：開始預熱。
3. 剛開機時，可以先把 `SV` 設成預計入豆前的目標溫度，例如 `160°C`。
4. 設定 `Air`、`Drum`，並決定 `Burner` 要手動控制，還是用 `AUTO Burner` 輔助預熱。
5. 預熱到目標狀態後按 `START`：開始記錄這一鍋。
6. 倒豆進烘豆機時按 `CHARGE`：記錄入豆。
7. 豆子轉黃、青味下降時按 `DRY END`：記錄乾燥結束。
8. 聽到穩定一爆時按 `FC START`：記錄一爆開始。
9. 決定下豆時按 `DROP`：記錄下豆。
10. 馬上按 `ON COOLING`：開始冷卻。
11. 冷卻完成後按 `COOL END`。
12. 最後按 `OFF`：停止記錄。

剛開始最重要的是按準這幾個事件：

- `START`
- `CHARGE`
- `DRY END`
- `FC START`
- `DROP`
- `COOL END`
- `OFF`

其他按鈕可以慢慢補。

## 上方控制按鈕

| 按鈕 | 白話意思 | 什麼時候用 |
| --- | --- | --- |
| `CONTROL` | 讓 Artisan 控制烘豆機 | 連線成功後、要開始操作 Kaleido 時 |
| `START HEATING` | 開始加熱預熱 | 烘豆前預熱時 |
| `START` | 開始記錄曲線 | 預熱完成、準備正式開始這一鍋時 |
| `OFF` | 停止這次記錄或關閉目前烘焙流程 | 一鍋完成、冷卻後 |

注意：`START` 不是入豆，`CHARGE` 才是入豆。

一個常見錯誤是預熱時就按了 `CHARGE`。正確概念是：

- `START`：這一鍋的紀錄開始。
- `CHARGE`：豆子真的進機器。

## 左側滑桿

| 名稱 | 意思 | 初學者先怎麼理解 |
| --- | --- | --- |
| `Air` | 風量 | 影響排煙、排濕、熱帶走多少 |
| `Burner` | 火力 / 加熱功率 | Kaleido 是電熱機，但 Artisan 介面用 Burner 表示加熱輸出 |
| `SV` | 目標設定值，常見於自動控制或 PID | 可以當作預熱目標，例如先設 `160°C`；是否會實際自動控火，要看是否啟用 `AUTO Burner` 以及目前的 Kaleido / Artisan 設定 |
| `Drum` | 滾筒轉速 | 影響豆子翻動與受熱均勻度 |

剛開始建議先用下方快捷按鈕調整，不要在烘焙中頻繁大幅拖滑桿。

`PID` 可以先理解成「機器或軟體為了接近目標溫度，自動調整輸出強弱的控溫方式」。對初學者來說，先知道它會嘗試追目標溫度就夠了，不需要一開始研究參數。

2026-07-05 的空機測試顯示，目前這台 Kaleido M1 搭配 Artisan 時，`SV` 不能當成無關緊要的欄位。當 `SV` 是 `0°C` 或很低時，只把 `Burner` 拉高，ET / BT 不一定會上升；把 `SV` 拉高後，溫度才開始跟著上升。正式烘豆時，入豆後不要讓 `SV` 長時間停在預熱用的低目標，詳見 [Kaleido M1：SV、Burner、Air 的關係測試筆記](kaleido-m1-sv-burner-air-relationship.md)。

## 剛開機預熱：SV 與 AUTO Burner 怎麼想

`SV` 是 Set Value，也就是「你希望機器接近的目標值」。如果這一鍋計畫用 `160°C` 左右入豆，剛開機預熱時可以先把 `SV` 設成 `160°C`，讓它成為預熱目標。

但要注意，`SV = 160°C` 只代表你設定了一個預熱目標；它不等於整鍋自動烘焙，也不能保證豆子進去後 `BT`（Bean Temperature，豆溫）會維持在 `160°C`。入豆後，室溫生豆會吸熱，豆溫先下降再回升，這是正常現象。

比較保守的預熱理解是：

1. 先按 `CONTROL`，確認 Artisan 能控制 Kaleido。
2. 按 `START HEATING` 開始加熱。
3. 設定 `Drum`，讓滾筒保持轉動；設定 `Air`，讓機器有基本排風。
4. 把 `SV` 設成預計入豆前目標，例如 `160°C`。
5. 如果目前介面與機器設定支援，可以按 `AUTO Burner`，讓火力自動嘗試追 `SV`。
6. 觀察 `Burner` 是否真的會自己變化，以及溫度是否穩定接近目標。
7. 接近目標後不要立刻入豆，先讓機器熱狀態穩定一段時間，再按 `START`、入豆並按 `CHARGE`。

初學者最容易誤會的是：按了 `AUTO Burner` 之後，機器不一定會自動幫你安排 `Air` 和 `Drum`。通常可以先把它理解成「自動調整加熱輸出」；`Air` 風量和 `Drum` 滾筒轉速仍然要自己設定，除非你另外設定了對應的自動控制或警報規則，而且已經確認你的 Kaleido / Artisan 設定真的支援。

所以實務上可以這樣做：

- 預熱階段：可以用 `SV = 160°C` 搭配 `AUTO Burner`，把它當成穩定入豆前溫度的輔助。
- 正式烘焙階段：初學者建議先切回 `MANUAL Burner`，用手動火力、固定幾段風量來學曲線反應。
- 如果要全程用自動控制：務必記錄當時是否為 `AUTO Burner`、`SV` 設多少、`Air` / `Drum` 是否手動固定，否則回顧時會不知道到底是什麼操作造成曲線變化。

第一次測試時，請盯著畫面確認三件事：

- `Burner` 有沒有因為 `AUTO Burner` 自己升降。
- `Air` 和 `Drum` 有沒有維持在你設定的數值。
- 溫度接近 `SV` 後，是穩定在附近，還是一直大幅上下震盪。

如果這三件事還沒確認，就先把 `AUTO Burner` 當成「待確認功能」，不要把它當成可以放心交給機器的自動烘焙模式。烘豆時仍然要全程在旁邊看顧機器，注意煙霧、銀皮與高溫安全。

## 右側溫度與曲線讀值

| 名稱 | 全名 | 白話意思 | 初學者要看什麼 |
| --- | --- | --- | --- |
| `BT` | Bean Temperature | 豆溫 | 烘焙進程最主要看的溫度 |
| `ET` | Environment Temperature | 環境溫 / 爐溫 | 反映烘焙環境熱量狀態 |
| `ΔBT` | Delta BT / Rate of Rise | 豆溫上升速度 | 看升溫是否太快、太慢或快失速 |
| `AT` | 依設定而定 | 額外溫度通道 | 需要對照 Kaleido / Artisan 設定確認 |

`ΔBT` 常被叫做 `RoR`，也就是 Rate of Rise。它不是豆溫本身，而是豆溫上升的速度。

## 下方藍色事件按鈕

這一排主要是「幫曲線做標記」。按下去不一定代表機器會自動做什麼，而是告訴 Artisan：這個時間點發生了某件重要事情。

| 按鈕 | 中文意思 | 什麼時候按 | 初學者判斷方式 |
| --- | --- | --- | --- |
| `CHARGE` | 入豆 | 生豆倒進烘豆機時 | 豆子真的進機器那一刻 |
| `DRY END` | 乾燥結束 | 豆子轉黃、草味下降時 | 顏色由綠轉黃，味道從青草轉成穀物、麵包感 |
| `FC START` | 一爆開始 | 聽到穩定一爆時 | 不只一兩聲零星聲音，而是開始連續爆裂 |
| `FC END` | 一爆結束 | 一爆聲明顯變少時 | 初學階段可以先不強求精準 |
| `SC START` | 二爆開始 | 聽到較細碎、密集的二爆聲時 | 中深焙以上才比較常用 |
| `SC END` | 二爆結束 | 二爆結束時 | 初學階段通常先不用 |
| `DROP` | 下豆 | 豆子離開烘焙室時 | 按下下豆、豆子進冷卻盤那一刻 |
| `COOL END` | 冷卻結束 | 冷卻完成時 | 豆子降到接近室溫、可以安全保存時 |

初學者最重要的是：`FC START` 和 `DROP` 要盡量按準。因為這兩個點會影響你之後計算一爆後發展時間，以及判斷這鍋是偏淺、偏中還是偏深。

## 下方火力快捷按鈕

紅色 `Burner` 按鈕是火力快捷鍵，例如：

- `100% Burner`
- `90% Burner`
- `80% Burner`
- `70% Burner`
- `60% Burner`
- `50% Burner`
- `40% Burner`
- `30% Burner`
- `20% Burner`
- `10% Burner`
- `OFF Burner`

白話理解：

- 百分比越高，熱量輸入越大。
- 火力越大，豆溫上升通常越快。
- 火力太大，容易外熟內生、表面焦、RoR 過高。
- 火力太小，可能升溫無力、拖太久、風味悶或空。

畫面上的：

- `++ Burner`：火力加一格。
- `-- Burner`：火力減一格。
- `AUTO Burner`：自動火力控制；通常是讓加熱輸出嘗試追 `SV`，實際行為要以你的 Kaleido / Artisan 設定為準。
- `MANUAL Burner`：手動火力控制；你按多少火力，機器就照那個設定輸出。

初學階段建議正式烘焙時先用 `MANUAL Burner`。你需要先知道「我按了什麼，曲線怎麼反應」，不要太早把判斷交給自動控制。預熱階段若要用 `AUTO Burner` 輔助穩溫，可以，但要先確認它只是在幫你追目標溫度，不代表 `Air`、`Drum` 或整條烘焙曲線都會自動處理。

## 下方風量快捷按鈕

藍色 `Air` 按鈕是風量快捷鍵，例如：

- `60% Air`
- `50% Air`
- `40% Air`
- `30% Air`
- `20% Air`
- `OFF Air`

白話理解：

- 風量會影響排煙與排濕。
- 風量也會帶走熱，所以不是越大越好。
- 風太小，煙味、悶味、銀皮排出可能較差。
- 風太大，熱被帶走太多，可能讓升溫變慢。

畫面上的：

- `++ Air`：風量加一格。
- `-- Air`：風量減一格。

初學階段可以先用固定幾段風量，不要每分鐘都改。這樣比較容易看出火力調整的效果。

## 下方滾筒快捷按鈕

棕色 `Drum` 按鈕是滾筒轉速快捷鍵，例如：

- `100% Drum`
- `90% Drum`
- `80% Drum`
- `70% Drum`
- `60% Drum`
- `50% Drum`
- `OFF Drum`

白話理解：

- 滾筒讓豆子持續翻動，避免局部受熱。
- 轉速會影響豆子翻滾狀態與受熱接觸。
- 烘焙中不要隨便按 `OFF Drum`。

剛開始可以先用固定轉速，不要把火力、風量、轉速三個變因同時改太多，否則回顧時很難判斷是哪個操作造成結果改變。

## 建議的初學操作心法

第一階段不要追求複雜曲線，先建立紀錄習慣：

- 每一鍋都按準 `CHARGE`、`DRY END`、`FC START`、`DROP`。
- 每次火力或風量調整，都讓 Artisan 留下事件或至少在筆記中記錄。
- 不要一次改太多變因。
- 不要離開正在運轉的烘豆機。
- 一鍋的結果不要單看曲線，要搭配杯測或沖煮後的風味回饋。

## 一鍋烘焙的簡化流程

```text
連線 Kaleido
  -> CONTROL
  -> START HEATING
  -> 設定 Air / Drum
  -> 設定 SV，例如 160°C
  -> 選擇手動 Burner，或用 AUTO Burner 輔助預熱
  -> 預熱並確認熱狀態穩定
  -> START
  -> CHARGE
  -> DRY END
  -> FC START
  -> DROP
  -> ON COOLING
  -> COOL END
  -> OFF
```
