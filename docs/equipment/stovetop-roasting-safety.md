# Kaleido M1 放在玻璃瓦斯爐台上的安全須知

這篇是給第一次烘豆前看的安全檢查表。你的實際操作條件是：

- 烘豆機是 Kaleido M1，屬於插電加熱的電熱式烘豆機。
- 瓦斯爐使用天然氣，但烘豆時不使用瓦斯爐火。
- 烘豆機預計放在玻璃瓦斯爐台上，位置是兩個爐架中間的玻璃檯面，不是放在爐架上。
- 主要目的是利用上方抽油煙機，把烘豆煙、熱氣與氣味排出去。

先把最重要的話寫在前面：

- 烘豆全程不能離開機器。
- Kaleido M1 是插電加熱，瓦斯爐火全程不可開啟；瓦斯爐台只能當作排煙位置，不是熱源。
- 如果家中瓦斯爐有獨立瓦斯閥，烘豆前先關閉，避免誤開。
- 放在兩個爐架中間的玻璃面，比放在爐架上穩，但仍要確認玻璃檯面承重、耐熱、點壓與刮傷風險。
- 抽油煙機可以幫忙排煙，但不能取代消防安全、通風、一氧化碳警報器與人在旁邊看顧。

## 這個場景有哪些風險

在瓦斯爐台上烘豆，風險比一般桌面操作多，因為你把幾件事情放在同一個區域：

| 風險 | 為什麼重要 | 初學者要先做到 |
| --- | --- | --- |
| 高溫 | 烘豆機、豆子、銀皮、排煙口都可能很燙 | 不碰熱金屬，不把塑膠、紙、布放旁邊 |
| 明火或瓦斯 | 這台 M1 不需要瓦斯火；真正風險是爐火誤開或瓦斯外洩 | 確認爐火與瓦斯閥關閉 |
| 玻璃爐台受損 | M1 約 10kg，長時間烘焙會有熱、震動與局部點壓 | 不拖拉機器，不墊可燃物，先查瓦斯爐說明書 |
| 煙與廢氣 | 烘豆會有煙、氣味、細小銀皮；家中仍有天然氣設備 | 開抽油煙機，補新鮮空氣，裝一氧化碳警報器 |
| 銀皮起火 | 銀皮很輕、很乾，接近高溫或火源容易燃燒 | 每鍋前清銀皮，準備金屬容器 |
| 用電負載 | 電熱烘豆機功率高，延長線或插座過載會發熱 | 插牆壁插座，不用延長線或排插 |
| 緊急處理不熟 | 初學者第一次遇到濃煙、跳電、火苗會慌 | 開始前先決定停止步驟與逃生路線 |

## 已確認：Kaleido M1 是插電式，不使用瓦斯火

這份文件以你的 Kaleido M1 為準，瓦斯爐只當排煙位置，不當熱源。

Kaleido M1 的原廠硬體手冊標示它是 `Electric heating, plug and play`，加熱功率約 `1000W`，機器淨重約 `10kg`，加熱元件表面瞬間可到 `500°C`。這代表安全重點是「插電設備 + 高溫機器 + 玻璃瓦斯爐台 + 排煙」，不是瓦斯火控火。

操作原則：

1. 瓦斯爐火保持關閉，烘豆中不要開火。
2. 如果家中瓦斯爐有獨立瓦斯閥，烘豆前先關閉。
3. 瓦斯爐旋鈕確認在 `OFF`，避免身體、電線或器材碰到旋鈕。
4. 烘豆機放在兩個爐架中間的玻璃面時，要完全放平、不可搖晃。
5. 電線要遠離爐口、烘豆機熱面、排煙管與水槽。
6. 烘豆機底部、側邊散熱孔與風扇不能被擋住。

## 放在玻璃瓦斯爐台中間可以嗎

比較務實的判斷是：可以考慮，但不要直接把它當成已確認安全。

你的想法有合理的地方：M1 的烘焙室與加熱區不在機器最底部，底部還有機體空間；放在兩個爐架中間的玻璃面，也比放在爐架上更平穩。真正要確認的是玻璃瓦斯爐台本身能不能承受這種用法。

玻璃爐台要注意：

- 承重：M1 約 `10kg`，加上操作時開關豆門、拉取樣棒、下豆與震動，不是單純靜態放著。
- 點壓：如果機器腳墊很小，重量會集中在幾個點，對玻璃比大面積平均受力更不友善。
- 刮傷：機器不能在玻璃面上拖拉；要移動就整台抬起來。
- 熱衝擊：玻璃怕局部冷熱差。烘豆機底部即使不是最高溫區，也可能讓某些區域長時間偏熱。
- 旋鈕誤觸：機器、電線或手臂不能碰到瓦斯爐旋鈕。
- 散熱：不要在 M1 底下墊毛巾、紙板、木板、塑膠板或厚矽膠墊，這些可能悶熱、變形或起火。

第一次正式烘豆前，建議先做一次短時間空機測試：

1. 瓦斯閥關閉，瓦斯爐旋鈕全部在 `OFF`。
2. M1 放上玻璃面後，用手輕推確認不晃、不滑。
3. 開抽油煙機，確認排煙方向會往上，不會把熱氣吹回機器或電線。
4. 先預熱短時間，不放豆，人在旁邊看。
5. 停機冷卻後，用手背靠近檢查玻璃面、機器腳座附近、旋鈕附近是否異常燙；不要直接摸高溫位置。
6. 只要聞到塑膠味、焦味、瓦斯味，或玻璃附近異常發熱，先不要正式烘。

如果你的玻璃瓦斯爐說明書明確寫不能放重物、不能當置物台、不能放高溫物品，這個位置就不適合直接放 M1。

## 烘豆前：環境檢查

開始前先把瓦斯爐台當成「高溫工作區」清空。

### 爐台周圍

- 把紙巾、抹布、塑膠袋、包裝袋、酒精噴瓶、咖啡豆袋、木砧板、油瓶移開。
- 窗簾、廚房紙巾架、調味料架不要靠近爐台。
- 爐台上不要同時煮東西。
- 不要讓小孩、寵物或其他人從你身後穿過。
- 手機可以放旁邊，但不要讓充電線跨過爐台。

原則是：會燒、會融、會被熱風吹動的東西，都不要放在烘豆機附近。

### 抽油煙機與通風

抽油煙機要先確認三件事：

1. 有真的排到室外，不只是室內循環濾網。
2. 濾網不要厚厚一層油垢，油垢遇到高溫與火星會增加火災風險。
3. 抽風開到足以把煙往上帶走，但不要讓烘豆機本體不穩，也不要讓熱氣和煙倒灌到電線或控制區。

烘豆前先開抽油煙機，並打開一扇窗或留一個進風口。抽油煙機把空氣抽出去時，室內需要有新鮮空氣補進來，排煙才會順。

如果你聞到瓦斯味、抽油煙機沒有效果、室內很快變煙霧瀰漫，立刻停止，不要硬烘完。

### 一氧化碳

一氧化碳（Carbon Monoxide, CO）是看不到、聞不到的有毒氣體。只要有燃燒設備，通風不良或燃燒不完全時就有風險。天然氣瓦斯爐、瓦斯熱水器、暖氣、發電機都屬於要注意的設備。

建議：

- 家裡裝一氧化碳警報器，至少放在睡眠區外，也建議廚房附近有警示能力。
- 烘豆時保持通風，不要把廚房門窗全部關死。
- 如果警報器響、頭痛、暈、想吐、嗜睡、意識不清，立刻離開室內，到新鮮空氣處，再打 `119`。

不要用「我有開抽油煙機」取代一氧化碳警報器。抽油煙機故障、風向回灌、濾網堵塞或補氣不足時，人不一定會立刻察覺。

## 烘豆前：設備與用電檢查

### 插座與電線

電熱烘豆機通常功率不低，用電要保守。

- 優先插牆壁固定插座。
- 不使用延長線、捲線盤、排插或多孔轉接頭。
- 不把三孔接地插頭硬插進兩孔插座，也不要剪掉接地腳。
- 插頭要插到底，不能鬆鬆的。
- 電線不能碰到瓦斯爐口、熱金屬、烘豆機排煙口、水槽或濕抹布。
- 如果插頭、插座、電線有發熱、焦味、變色、鬆動，停止使用。

如果烘豆機一開就跳電，或同一個迴路還接著電鍋、烤箱、微波爐、電熱水壺，不要硬撐。這代表負載可能太高，需要重新安排插座或請水電檢查。

### 烘豆機本體

每一鍋前都檢查：

- 滾筒或攪拌結構能正常轉動。
- 風扇能正常排風。
- 銀皮收集處已清空。
- 排煙路徑沒有堵住。
- 溫度線、USB 線、電源線沒有碰到熱面。
- 冷卻盤或冷卻流程已準備好。
- 下豆後豆子可以快速冷卻，不會堆在塑膠袋或紙盒裡。

烘豆機如果有異常摩擦聲、風扇不轉、滾筒不轉、控制器失靈、冒出塑膠焦味，這一鍋不要開始。

## 建議準備的安全用品

至少準備這些東西，而且要放在伸手可及、但不會被火擋住的位置：

| 用品 | 用途 | 注意 |
| --- | --- | --- |
| 住宅用 ABC 乾粉滅火器 | 可處理常見固體、油類、部分電氣火災 | 先看說明，確認壓力正常 |
| 滅火毯 | 小範圍火焰或保護撤離時可用 | 不要用來蓋住仍插電且大量冒煙的設備太久 |
| 金屬托盤或金屬桶 | 放熱銀皮、熱零件、異常豆子 | 不用塑膠桶或紙箱 |
| 隔熱手套 | 移動熱部件 | 要乾燥，濕手套導熱更快 |
| 手電筒 | 跳電時照明 | 不要靠手機當唯一照明 |
| 一氧化碳警報器 | 偵測 CO 風險 | 定期測試與更換電池 |
| 煙霧警報器 | 提早發現火災 | 不要因為烘豆會冒煙就永久拆掉 |

滅火器只適合在火勢很小、你知道怎麼用、背後有退路時使用。只要不確定，先撤離，再打 `119`。

## 烘豆中：你要盯什麼

第一次烘豆時，不要一邊做飯、洗碗、接電話或離開廚房。

烘豆中固定觀察：

- 煙有沒有被抽油煙機帶走。
- 瓦斯爐旋鈕是否保持關閉。
- 電線是否有滑動、碰熱或被拉扯。
- 銀皮是否大量堆積或飛到熱源附近。
- 烘豆機聲音是否突然變大、卡住或停止轉動。
- 有沒有塑膠味、電線焦味、瓦斯味。
- 家中警報器是否響起。

如果你覺得「怪怪的」，不要說服自己撐到下豆。初學者第一優先是停機安全，不是把曲線烘完。

## 停止操作的判斷

遇到下面任何一種情況，直接停止烘豆流程：

- 聞到瓦斯味。
- 電線、插頭、插座發熱或有焦味。
- 烘豆機冒出異常濃煙，不像正常咖啡煙。
- 銀皮、豆子或機器內部出現火苗。
- 抽油煙機突然停止。
- 風扇或滾筒停止。
- 一氧化碳警報器或煙霧警報器響。
- 你需要離開廚房。

停止時先做安全動作：

1. 正常情況下瓦斯火不應該開；如果爐火被誤開，先關火，安全可行時再關瓦斯閥。
2. 停止烘豆機加熱。
3. 如果是電氣異常，且插頭位置安全、沒有火與濃煙擋路，再拔插頭。
4. 保持退路，不要站在火與出口之間。
5. 火勢、煙或瓦斯味無法控制時，離開室內並打 `119`。

## 如果真的起火

先判斷火在哪裡。

### 豆子或銀皮在機器內起火

- 不要急著打開烘豆機。打開可能讓更多氧氣進去，火勢變大。
- 停止加熱。
- 依原廠手冊處理風扇、滾筒與關機流程；如果沒有把握，先撤離。
- 不要往插電設備或高溫機器裡倒水。
- 如果火很小、侷限、你有退路且會用滅火器，才考慮滅火器。
- 火勢變大、濃煙變多、你無法判斷時，離開並打 `119`。

### 爐台或周圍物品起火

- 如果瓦斯爐火被誤開，且安全可行，先關瓦斯爐火。
- 不要用水潑向油火、電器或插座。
- 可以用滅火毯或合適滅火器處理小範圍火焰。
- 火勢超過你能控制的範圍，立刻撤離。

### 聞到瓦斯味但還沒起火

- 不要開關任何電器或燈。
- 不要插拔插頭。
- 不要點火測試。
- 如果安全可行，關閉瓦斯閥並打開窗戶。
- 離開室內，到室外再打電話求助。

## 烘完後

烘完不代表安全結束。豆子、銀皮、烘豆機和排煙路徑還是熱的。

1. 讓豆子完全冷卻後再裝袋。
2. 熱銀皮先放金屬容器，不要直接倒進垃圾桶。
3. 抽油煙機繼續開一段時間，直到煙味明顯下降。
4. 確認瓦斯爐旋鈕與瓦斯閥關閉。
5. 確認烘豆機已停止加熱，冷卻完成後再拔插頭。
6. 檢查爐台、抽油煙機濾網、周邊檯面是否有銀皮或焦屑。
7. 記錄這次是否有煙太大、抽風不足、插座發熱、銀皮堆積等問題。

如果第一次烘完覺得家裡煙味很重，下一鍋不要只調烘焙曲線，先改善排煙與通風。

## 第一次烘豆前的保守版本

如果你完全沒有烘過，第一鍋建議採用保守流程：

1. 白天烘，不要半夜烘。
2. 家裡至少另一個人知道你在烘豆。
3. 批次不要直接上限，先用計畫中的小批次。
4. 烘豆前做一次空機或短時間測試，確認風扇、滾筒、加熱、Artisan 連線正常。
5. 先確認「怎麼停機」，再開始想「怎麼烘好喝」。
6. 不追求深焙。越接近二爆，煙、油、銀皮與起火風險通常越高。
7. 烘完後先整理安全問題，再整理風味問題。

## 不適合開始烘豆的情況

只要有以下任一狀況，這一鍋先不要烘：

- 烘豆機放在瓦斯爐上會晃。
- 不確定瓦斯爐火會不會被誤開。
- 抽油煙機排不出去，或濾網油垢很厚。
- 必須使用延長線或排插才插得到電。
- 插座、插頭、電線曾經發熱。
- 家裡已經聞得到瓦斯味。
- 玻璃爐台已經有裂痕、缺角、明顯刮傷或固定不穩。
- 玻璃瓦斯爐說明書明確禁止放重物、熱物或把爐面當置物平台。
- 沒有任何滅火設備，也不知道最近的出口路線。
- 你需要中途離開廚房。

安全上，少烘一鍋沒有損失；硬烘一鍋出事，代價很高。

## 待確認清單

這些資訊未來可以補進專案，讓安全須知更精準：

- 你的 Kaleido M1 版本與對應原廠安全手冊。
- 玻璃瓦斯爐品牌型號、玻璃面板承重限制、是否允許作為置物平台。
- M1 放上玻璃爐台後，四個腳墊是否完整接觸、是否有點壓或晃動。
- 烘豆機功率、插座額定電流、家中迴路配置。
- 抽油煙機是否排到室外，風量是否足夠。
- 家中是否已有煙霧警報器與一氧化碳警報器。
