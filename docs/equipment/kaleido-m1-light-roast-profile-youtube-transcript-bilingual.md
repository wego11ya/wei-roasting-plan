## 烘焙重點整理

這支影片最值得放進你第一鍋參考的是：

- 小滾筒不要硬塞滿，影片作者從 `180g` 改成 `150g`，讓豆子膨脹後仍有空間翻動。
- 高密度、大顆豆需要足夠前段熱量，否則熱進不到豆芯。
- 含水量偏低時，前段不能只是一味大火硬衝，中後段要逐步降火。
- 使用較高風量是為了增加對流、排濕、排煙，但風量太大也可能把熱帶走。
- 一爆前後要小心 `flick` 和 `crash`，不要只照表操課，要看 RoR 反應微調。
- 影片的下豆策略是非常淺焙，重點在花香、果汁感、沒有烘焙味，也沒有明顯酸澀或苦味。

跟你的衣索比亞南西寶水洗藝伎相比：

- 可以參考 `150g`、`160°C` 入豆、約 `4:30` 乾燥結束、約 `8:10` 一爆、約 `9:20` 下豆這個骨架。
- 但你的豆子是衣索比亞高海拔水洗藝伎，不是厄瓜多 Cidra，所以不要把影片曲線當成標準答案。
- 第一鍋要優先追求酸甜平衡，不建議一開始就極限淺焙。
