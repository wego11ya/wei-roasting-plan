## 當下觀察

- 預熱到 SV 後等了五分鐘以上，但 ET 和 BT 差距仍然比 reference 圖大。
- 入豆後約 `0:40` 就到回溫點，速度很快。
- 入豆後約 `1:01` 才把 SV 拉到 `215°C`，這會讓操作節奏變亂；更大的問題是 `102g` 小批次仍用 `60%` Burner 推到 Dry End 附近。
- Dry End 實際可能比手動標記更早，因為當時沒有頻繁取樣觀察豆色。
- FCs 在 `4:14`，而且是一段零星但連續的爆裂，不像 reference 那種比較後段才來、較集中可辨識的一爆。
- FCe 標記不確定，這對初學階段正常；下一鍋先專注 FCs 和 Drop。
- 五分鐘出頭 RoR 往上翹，當下已經來不及只靠降火修正。

- 剛烘完沖煮有喝到桃子的甜美，但苦味重；後續即使磨很粗仍然苦。

## 主要判讀

### 1. 這鍋最大原因是批次太小但熱量太強

Reference 是 `150g`，這鍋實際是 `102.1g`。豆量少，豆子能吸收的熱量少，同樣的 SV、Burner 和預熱狀態會讓豆子升溫更快。

所以這鍋不能只歸因於事件按慢。事件按慢會讓數字有誤差，但曲線本身已經顯示整鍋節奏太快：

```text
Dry End 3:13
FCs 4:14
Drop 6:57
```

對這支水洗藝伎想要花果與甜感的方向，`FCs 4:14` 太早，前段與中段都被壓縮。

### 2. SV 需要下修

這鍋入豆後 SV 拉到 `215°C`，搭配 `60%` Burner 對 `102g` 太猛。第二鍋如果仍烘 `100g` 左右，先把設定改成：

```text
預熱 SV：150°C
入豆後 SV：200°C
起始 Burner：50%
```

目標先放在一爆開始不要再早於 `6:30`，最好拉回 `7:30-8:30`。

### 3. 一爆後拖太久，苦味合理

這鍋 FCs 到 Drop 是 `2:43`，DTR 是 `39.2%`。這對偏淺焙目標太長，搭配 `15.67%` 失重率，苦味重很合理。

下一鍋先用簡單規則：

```text
按 FC START 後開始算
1:00-1:15 之間準備下豆
先不要等 FC END
```

### 4. ET / BT 差距不能直接拿來跟 reference 絕對比較

ET（Environment Temperature，環境溫）和 BT（Bean Temperature，豆溫）是不同探針位置，不一定會貼近。你這鍋後段 BT 高於 ET 很多，主要是因為：

- FC 前後你降 Burner、加 Air，ET 會比較快掉。
- 豆子和滾筒仍有熱慣性，BT 還會繼續上升。
- 一爆期間豆子內部反應會讓 RoR 變得不穩。
- 批次小，探針接觸豆子的狀態和氣流影響都更明顯。

這目前還不能直接判定感測器壞。第二鍋要用同一台機器、同一批次量、同一個 Artisan 設定，再看 ET / BT 的相對變化是否更穩。

## 第二鍋要驗證什麼

第二鍋只驗證三件事：

1. `100g` 左右小批次，把預熱 SV 降到 `150°C`、入豆後 SV 降到 `200°C`、起始 Burner 降到 `50%`，能不能把 FCs 拉回 `7:30-8:30`。
2. FCs 後只發展 `1:00-1:15`，能不能把苦味壓下來。
3. Artisan 的 Roast Properties 要先填正確生豆重量，烘後補熟豆重量，讓失重率可以直接回顧。

## 下次操作提醒

- 烘豆前在 Artisan `Roast -> Properties` 填入豆重量。
- 烘完補熟豆重量。
- `Config -> Phases` 把 Phases LCD 設成比較好判斷的時間顯示，Finishing 的 `Phases LCDs All` 保持打勾。
- 聽到穩定一爆後按 `FC START`，同時按手機碼表。
- 這一輪先不管 FCe 標得準不準，先把 FCs 和 Drop 做準。
