# 第一鍋烘焙計畫：317 跳舞山羊 西阿希南西寶 G1 水洗藝伎 / 2026-07-01

## 目標

這一鍋的目標是做一個偏淺焙、乾淨、花果調明顯，但不要尖酸或草青的版本。

希望喝到的方向：

- 前段香氣：野薑花、白花、檸檬皮。
- 中段風味：桃子、紅蘋果、明亮但不刺的果酸。
- 後段甜感：棉花糖、白糖水感，口感乾淨。

烘焙策略：

- 不追求極淺焙。第一鍋先以「酸甜平衡」優先，保留水果調性，同時降低烘不熟、空洞、草味的風險。
- 總烘焙時間抓在約 `9:05-9:35`。
- 一爆開始後發展約 `1:05-1:20` 下豆。
- 預估下豆豆溫抓 `192-194°C`，或以「一爆開始 BT + 約 `3-5°C`」判斷。Kaleido M1 與 Artisan 的感測設定可能不同，實際要以一爆時間、豆色、氣味與杯測結果一起判斷。

術語先記這幾個：

| 名稱 | 白話意思 | 這鍋怎麼用 |
| --- | --- | --- |
| 入豆溫（charge temperature） | 生豆倒進烘豆機時，機器顯示的豆溫或環境熱狀態 | 本計畫先用約 `155-158°C` 入豆 |
| 設定值（Set Value, SV） | 控制器的目標設定值，不等於豆溫一定會到的溫度 | 預熱時設 `155-158°C`，入豆後設 `215°C` 作為熱量供給目標 |
| 回溫點（turning point） | 入豆後 BT 會先下降，降到最低後重新開始上升的點 | 這鍋希望在 `0:40-0:55` 出現 |
| 豆溫（Bean Temperature, BT） | 探針讀到的豆子附近溫度 | 用來標記回溫、一爆與下豆 |
| 環境溫（Environment Temperature, ET） | 烘焙室熱環境的溫度 | 不同機器設定差很多，第一鍋先輔助觀察 |
| 升溫率（Rate of Rise, RoR） | 豆溫每分鐘上升多少度 | 目標是乾燥後逐步下降，不要一爆前後突然暴衝或歸零 |
| 梅納反應（Maillard reaction） | 豆子變黃後到一爆前，香氣與甜感逐漸形成的反應 | 這段不要拖太久，也不要熱量不足 |
| 一爆（First Crack, FC） | 豆子內部水蒸氣壓力讓結構裂開，聽起來像爆米花較低沉的爆裂聲 | 聽到連續爆裂才按 `FC START` |
| 發展時間（development time） | 一爆開始到下豆的時間 | 這鍋抓約 `1:10`，讓酸轉甜一點但不烘太深 |
| 發展比例（Development Time Ratio, DTR） | 發展時間占總烘焙時間的比例 | 這鍋約 `12-14%`，只當參考，不要單看比例決定下豆 |

### 生豆判讀

這支豆子的密度 `842 g/L` 很高，代表豆子結構扎實，通常需要足夠的前段熱量，熱才比較進得去豆芯。

含水量 `9.9%` 偏低一點。這代表它不像高含水豆那樣需要很長時間排水，但也比較不適合前段用過高火力硬衝，否則表面容易先熟、甚至出現焦點或粗糙感。

失重率會高於含水量是正常的。生豆含水量 `9.9%` 只是在烘焙前量到的水分比例；烘焙失重不只包含水分，還包含銀皮脫落、二氧化碳、揮發性香氣物質、少量有機物分解，以及烘焙反應中產生後又被帶走的水氣。也就是說，`100g` 生豆即使含水不到 `10g`，烘完仍可能少掉 `11-12g`。

初學者可以先這樣記：

- 生豆含水量：烘焙前，豆子裡原本有多少水。
- 烘焙失重率：烘焙後，整顆豆子總共少了多少重量。
- 兩者相關，但要分開看。

水洗處理法通常風味比較乾淨，這鍋要避免兩件事：

- 烘太短或發展不足：容易檸檬酸很尖、草青、空洞。
- 烘太久或一爆後火力太重：花香與桃子會變鈍，甜感可能變成焦糖或烘焙味。

| 預熱入豆溫 | `155-158°C` |
| SV 設定 | 預熱 `155-158°C`；入豆後先調到 `215°C` |
| 滾筒 | 固定 `90%`，參考影片設定，烘焙中不要改 |

這次改成 `100g`，原因是：

- Kaleido M1 是小滾筒，豆子膨脹後空間會變小。
- 藝伎與高海拔豆在烘焙中可能膨脹明顯。
- `100g` 對第一鍋更保守，失敗成本較低，也比較方便觀察機器反應。

但 `100g` 和影片的 `150g` 不同。豆量變少後，豆子吸收的熱比較少，機器原本累積的熱更容易把曲線推太快。因此這份計畫不能只把 `150g` 改成 `100g`，也要同步降低入豆溫與火力。

安全提醒：

- 烘焙全程不要離開機器。
- 預先確認通風、排煙、集塵與銀皮清理。
- 一爆後煙與銀皮會增加，注意不要讓排煙受阻。

### SV 怎麼理解

SV 是 `Set Value`，可以先理解成「控制器想追的設定值」或「你給機器的熱量目標」。它和 BT、ET 是不同欄位，所以不代表豆溫或環境溫一定要真的到 `220°C`。

從截圖看，影片作者入豆前把 SV 放在約 `160°C`，讓機器穩在可入豆狀態；入豆後把 SV 拉到約 `220°C`，目的是讓電熱系統有足夠熱量往上推。可是實際烘焙時，他仍然一直用 Burner 火力與 Air 風量調整 RoR，最後 BT 只到約 `191.4°C` 就下豆。

這種「SV 高於實際 BT / ET」在電熱小型烘豆機或 PID 控制環境中可以出現。重點是你要知道 SV 控的是機器的加熱目標，不能把它當成咖啡豆要到達的目標溫度。業界大型瓦斯滾筒機通常比較常直接看火力、風門、BT、ET、RoR 與事件時間，不一定會用 SV 這個欄位來操作；自動化或電熱機則比較常看到 setpoint / SV 的概念。

這份計畫參考影片的 SV 骨架，但因為批次改成 `100g`，先用比較保守的 SV：預熱 `155-158°C`，入豆後 `215°C`。如果第一鍋明顯推不動，例如乾燥結束晚於 `4:45` 或一爆晚於 `8:35`，下一鍋再考慮把入豆後 SV 提到 `220°C`。

### Drum 和 Air 怎麼理解

影片的烘焙邏輯確實比較偏向「小批次、高風量、提高對流」。對流是熱空氣把熱帶進豆子裡；相對地，傳導是豆子接觸滾筒壁或豆子彼此接觸時得到熱。

Drum 固定 `90%` 的目的，是讓豆子翻動更頻繁，降低某些豆子長時間貼著熱表面的機會，讓受熱更平均。Kaleido M1 是電熱小滾筒，不太適合用大型瓦斯機的「直火」概念完全套用；更準確地說，高 Drum 會降低局部接觸受熱的風險，讓高 Air 的對流效果更明顯。

這份計畫跟著影片採用 `90%` Drum。Air 則從 `60%` 起步、轉黃後逐步提高到 `70-75%`。原因是 `100g` 批次很輕，如果一開始 Air 太高，可能把熱帶走太多，造成前段推不動；如果 RoR 很高或豆面出現焦點，下一鍋再把早段 Air 提高。

## 操作步驟

### 烘焙前準備

1. 生豆秤 `100g`。
2. 開啟排煙與通風。
3. 確認 Artisan 已連上 Kaleido M1。
4. 使用手動控制模式。
5. 預熱到入豆狀態穩定後再開始。

預熱建議：

- 目標入豆溫：BT 約 `155-158°C`。
- 預熱 SV：`155-158°C`。
- 預熱穩定時間：到達目標後再等約 `5-8` 分鐘，讓烘焙室整體熱狀態穩定，不只看探針數字。
- 如果 ET 或機器熱狀態波動很大，先不要入豆。
- 入豆後立刻把 SV 調到 `215°C`，但不要期待 BT 或 ET 真的到 `215°C`；後續仍用 Burner、Air 和 RoR 來控節奏。

### 第一鍋建議參數表

以下是起始版，火力與風量的數字是 Artisan / Kaleido M1 介面百分比，後續要依實際曲線調整。

| 時間或事件 | SV | Burner 火力 | Air 風量 | Drum 滾筒 | 操作目的 |
| --- | ---: | ---: | ---: | ---: | --- |
| 預熱穩定 | `155-158°C` | 依機器預熱狀態 | `60%` | `90%` | 讓 BT 接近 `155-158°C` 並穩定 |
| 入豆 `0:00` | 調到 `215°C` | `60-65%` | `60%` | `90%` | 100g 熱慣性較低，用高 Drum 降低局部接觸風險 |
| `2:00` 左右 | `215°C` | `55-60%` | `60%` | `90%` | RoR 開始建立後，先小幅降火 |
| `3:15` 左右 | `215°C` | `50-55%` | `60-65%` | `90%` | 避免乾燥末段衝太快，開始增加排濕 |
| 乾燥結束 `4:05-4:35` | `215°C` | `40-45%` | `65-70%` | `90%` | 進入甜感建立期，增加排濕與對流 |
| `5:30` 左右 | `215°C` | `35-40%` | `70%` | `90%` | 維持乾淨感，讓 RoR 緩慢下降 |

| 一爆前 `7:30-8:00` | `215°C` | `10-20%` | `70-75%` | `90%` | 看 RoR 微調，避免一爆前 crash 或 flick |
| 一爆開始後 | `215°C` | `5-15%` | `75%` | `90%` | 排煙排濕，讓一爆穩定走完前段 |
| 下豆 | 不再作為判斷依據 | 關火 | 冷卻最大風量 | 依冷卻流程 | 迅速冷卻，避免餘熱繼續烘深 |

### 一爆前後怎麼判斷

一爆前最容易出現兩種問題：

- Crash：一爆前後 RoR 掉太快，甚至接近 `0`。喝起來可能悶、空、甜感不足。
- Flick：一爆後 RoR 突然往上翹。喝起來可能粗糙、苦、烘焙味壓過花果。

臨場處理方式：

| 狀況 | 立刻怎麼做 | 下次怎麼修 |
| --- | --- | --- |
| `7:30` 後 RoR 掉很快、一爆還沒來 | 火力短暫加回 `25-30%`，不要一次加太多 | 中段不要太早降到低火 |
| 一爆已經開始，RoR 繼續暴衝 | 火力降到 `10%` 或更低，風量維持 `75%` | 下次一爆前更早降火 |
| 一爆很弱、零星、拖很久 | 不要急著下豆，至少讓一爆後走 `1:15` 左右 | 下次前段或中段給多一點能量 |
| 一爆很密集又豆色快速加深 | 一爆後 `1:00-1:05` 可以準備下豆 | 下次一爆前火力再低一點 |

### 下豆判斷

這鍋的最佳下豆點先抓：

- 一爆開始後 `1:10` 左右。
- 總時間約 `9:05-9:35`。
- BT 約 `192-194°C`。如果一爆開始在 `188-189°C`，可以優先看 `192-193°C`；如果一爆開始較晚或 BT 較高，才看 `193-194°C`。
- 豆色是均勻淺褐，不要進入明顯中焙色。
- 香氣從穀物、麵包轉成甜香、花果、乾淨煙少。

如果只能記一個規則：

> 聽到穩定一爆後，不要立刻下豆；先讓它發展約 70 秒，讓酸質變甜，但不要拖到烘焙味明顯。

| 檸檬很尖、像未熟水果、甜感少 | 發展不足，或一爆太弱 | 一爆後多 `10-15` 秒，或下豆溫提高 `1-2°C` |
| 有草青、穀物生味、口感空 | 熱沒有進豆芯，可能中段能量不足 | 下次中段火力不要降太快，FC 目標維持在 `7:55-8:30` |
| 花香乾淨但太薄 | 烘得很淺，甜感還沒建立 | 下次延長梅納反應期或發展 `10` 秒 |
| 桃子、蘋果、棉花糖明顯 | 接近目標 | 下一鍋只微調，不要大改 |
| 酸甜平衡但花香弱 | 烘得稍深或總時間偏長 | 下次早 `10` 秒下豆，或一爆後火力更低 |
| 苦、煙、焦、粗糙 | 一爆後熱量太高或排煙不足 | 下次一爆前提早降火，FC 後風量維持高一點 |

## 下次調整

第一鍋不要只用曲線判斷成敗，要用「曲線事件 + 杯測」一起看。

優先調整順序：

1. 先固定批次量 `100g`，不要第二鍋就改成 `150g` 或 `180g`。
2. 先看乾燥結束時間有沒有落在 `4:05-4:35`。
3. 再看一爆有沒有落在 `7:55-8:30`。
4. 最後才調整一爆後發展時間。

下一鍋的簡單修正規則：

| 第一鍋結果 | 第二鍋建議 |
| --- | --- |
| Dry End 晚於 `4:50`，FC 晚於 `8:40` | 入豆後 SV 提到 `220°C`，或前 `3` 分鐘火力提高 `5%` |
| Dry End 早於 `3:50`，豆面有焦點 | 入豆溫降到 `152-155°C`，或起始火力降到 `55-60%` |
| FC 很弱，喝起來酸尖草青 | 一爆前不要降到 `15%` 以下，發展延長到 `1:20` |
| FC 很猛，喝起來苦或花香消失 | 一爆前更早降火，FC 後 `1:00-1:05` 下豆 |
| 酸甜平衡、花果清楚 | 保留這個 profile，只做 `5-10` 秒級的小調整 |
