## 先看這支豆子的前提

這支是衣索比亞南西寶 G1 水洗藝伎。供應商風味目標是檸檬、野薑花、桃子、紅蘋果、棉花糖般甜感。

所以判斷這鍋時，不適合用中深焙邏輯。這支豆目前比較合理的方向是：

- 保留乾淨酸甜與花果調。
- 不讓烘焙味、焦苦、煙味蓋過豆子本身。
- 避免第一鍋那種 `102g` 小批次熱量太強、FCs 太早、發展太久。
- 先用同一台 Kaleido M1、同一批次量級、同一套 Artisan 設定比較，不把 BT 絕對值直接拿去跟其他機器比。

## 主要判讀

### 1. 第二鍋確實比第一鍋進步很多

這不是只有曲線看起來比較漂亮而已。數字也支持：

- FCs 從 `4:14` 拉到 `6:24`。
- 發展時間從 `2:43` 壓到 `1:05`。
- 失重率從 `15.67%` 降到約 `12.6%`。
- 目測烘焙度變淺，實際也有酸甜。

這代表第一鍋最大的問題，也就是「小批次熱量太強、發展太久」，已經被修正一大段。

### 2. 但第二鍋的前中段仍然比 reference 快

Reference 是 `150g`，第二鍋是約 `102g`。豆量不同，所以不能要求曲線完全一樣。

但方向上，第二鍋仍偏快：

| 項目 | Reference | 第二鍋 |
| --- | ---: | ---: |
| DE | 約 `4:34` | `3:45` |
| FCs | 約 `8:12` | `6:24` |
| DROP | 約 `9:22` | `7:29` |
| FCs -> DROP | 約 `1:10` | `1:05` |
| DTR | 約 `12.5%` | `14.5%` |

重點不是第二鍋發展期太短。發展期其實接近 reference。真正差異在 FCs 前：第二鍋太早到一爆，代表乾燥期與梅納反應期仍被壓縮。

### 3. Drop `7:29` 夠不夠，要拆成兩件事看

如果問「FCs 後有沒有發展夠？」  
這鍋 `1:05`、DTR `14.5%`，以這支水洗藝伎的淺焙目標來說，是合理的起點。

如果問「整鍋總時間夠不夠？」  
`7:29` 仍偏短，因為 FCs 在 `6:24` 就來了。對高密度水洗藝伎來說，這可能讓甜感、花香層次和果肉感還沒完全拉開。

所以判斷是：

```text
發展期時間可以先接受。
FCs 前的進程仍偏快，下一鍋優先把 FCs 往後拉，而不是單純把 FCs 後拖長。
```

### 4. Drop BT `175.2°C` 不能單獨說夠或不夠

Artisan 上的 BT 是探針讀值，不等於豆芯真實溫度。不同機器、探針位置、批次量和風量都會讓 BT 絕對值不同。

這鍋在同一台 Kaleido M1 上可以這樣看：

- 第一鍋 Drop BT `185.8°C`，失重 `15.67%`，苦味明顯。
- 第二鍋 Drop BT `175.2°C`，失重約 `12.6%`，目測淺焙，有酸甜。
- 第二鍋 FCs BT `168.9°C`，Drop BT `175.2°C`，一爆後增加約 `6.3°C`。

所以 `175.2°C` 對這台機器、這個批次量、這支豆來說，不是明顯不夠。剛烘完沒有明顯青草、生豆味或穀物生味，代表目前不把第二鍋判定為「下豆太早」或「未熟」。它至少已經足以進入可喝的淺焙區間。

第 `3-5` 天的正式杯測主要用來確認這個判斷：如果休息後仍然酸甜乾淨，就先不要提高 Drop；如果休息後才出現明顯青草、穀物、生豆味、空洞酸，才回頭考慮下豆條件太早或前中段發展不足。

### 5. 入豆 BT `149.9°C` 夠不夠

這鍋入豆 BT 約 `150°C`，TP 在 `0:42`，FCs 在 `6:24`。這代表熱量不是不夠，反而對 `102g` 批次仍然推得偏快。

下一鍋不建議因為總時間短就把入豆溫拉高。比較合理的是：

- 入豆 BT 先維持 `148-150°C`。
- 起始 Burner 可以從 `50%` 小降到 `45%`。
- 或在 `1:30-2:00` 更早降一段火力。
- 目標先把 DE 拉到 `4:00-4:20`，FCs 拉到至少 `7:00` 後。

### 6. 發展期 RoR crash 會不會影響風味

RoR 是升溫率（Rate of Rise），可以先理解成豆溫上升的速度。你這鍋整體 RoR 是慢慢下降，這是好事。

但發展期中後段確實有明顯下探。用 BT 粗估：

| 時間 | BT | 近似 30 秒 RoR |
| --- | ---: | ---: |
| FCs `6:24` | `168.9°C` | 約 `8.6°C/min` |
| `6:45` | `172.1°C` | 約 `9.8°C/min` |
| `7:00` | `173.2°C` | 約 `6.1°C/min` |
| `7:15` | `174.3°C` | 約 `4.4°C/min` |
| Drop `7:29` | `175.2°C` | 約 `4.1°C/min` |

這不是歸零或負 RoR，但下降幅度很明顯。可能原因是 FCs 後熱量降得偏多、ET 開始下降、再加上一爆後豆子釋放水氣與結構變化，讓 BT 上升速度掉下來。

它可能造成的風味影響：

- 甜感斷掉，喝起來有酸但不夠圓。
- 香氣變薄，花果比較不立體。
- 口感變空或尾段有乾澀。
- 如果 crash 很嚴重且拖很久，可能出現悶、焙烤不足或類似 baked 的鈍感。

但這鍋 crash 發生在接近 Drop 前，而且時間不長。加上你目前喝到有酸甜，不能直接判定它已經造成大問題。這件事要用第 `3-5` 天杯測確認。

### 7. 為什麼目測淺焙，但研磨太細會苦

這不一定代表烘得太深。可能有幾種原因：

| 可能原因 | 怎麼理解 |
| --- | --- |
| 萃取過度 | 淺焙也會苦，尤其磨太細、粉層堵住、尾段萃太久時 |
| 細粉多 | 研磨太細或磨豆機細粉多，容易帶出澀苦 |
| 甜感還沒休息出來 | 剛烘完或休息不足時，酸甜結構可能還沒穩定 |
| 前中段仍偏快 | FCs 前太快，可能讓酸甜分離，甜感不夠支撐時苦感更明顯 |
| 發展期 RoR 下探 | 可能讓尾段甜感或口感變薄，苦感比較突出 |

所以目前不要只用「磨到中烘研磨度才不苦」判斷烘焙失敗。先固定沖煮條件，再比較第 `3`、`5`、`7` 天。

建議記錄：

```text
同一配方下，太細會苦；接近平常中烘焙研磨度時，酸甜比較乾淨。
```

## DE 到底什麼時候標記

DE 是 Dry End，乾燥結束。Artisan 官方把乾燥期定義為加熱豆子到豆子轉黃；DE 到 FCs 是梅納反應期；FCs 到 Drop 是發展期。

初學者不要只看固定 BT。Kaleido M1 小批次的 BT 會受探針接觸、風量、豆量影響。比較穩的判斷是「豆色 + 氣味 + 表面狀態」一起看：

| 觀察 | 還沒到 DE | 接近 DE / 可以標 DE |
| --- | --- | --- |
| 顏色 | 還有明顯綠、灰綠、斑駁 | 大多變黃、黃褐、米黃色 |
| 氣味 | 青草、豆腥、青豌豆、濕生味 | 穀物、麵包、玉米、乾草 |
| 表面 | 看起來濕、緊、綠感重 | 表面比較乾，銀皮開始明顯 |
| 聲音 | 通常無明顯聲音 | 通常仍無一爆聲 |

實作上可以先用這個規則：

```text
當 80% 以上豆子已經不是綠色，氣味從青草轉成穀物/麵包/玉米，就按 DE。
如果不確定，不要追求絕對準；每鍋用同一個標準，才有比較價值。
```

這鍋 DE 標在 `3:45`。如果當時豆色只是剛開始轉黃，實際 DE 可能標早了 `15-30` 秒。這不會推翻整鍋判讀，但會影響乾燥期與梅納期比例。

## FCs、Drop 時間與 BT 對風味的影響

### FCs 時間

FCs 是一爆開始（First Crack Start）。它大致反映 FCs 前的熱量與反應速度。

| FCs 狀況 | 可能風味 |
| --- | --- |
| 太早 | 香氣亮但甜感不夠，酸甜分離，可能有尖酸、草青、口感薄 |
| 適中 | 酸甜比較平衡，花果清楚，甜感能接住酸 |
| 太晚 | 可能更圓、更甜，但若拖太久也可能悶、香氣鈍 |

對這支豆，下一鍋可以先把 FCs 目標從原本 `7:30-8:30` 放寬成：

```text
先努力拉到 7:00-7:45
不要一次追 reference 的 8:12
```

### Drop 時間

Drop 是下豆時間。它決定整鍋最後停在哪裡，但要跟 FCs 一起看。

| Drop 狀況 | 可能風味 |
| --- | --- |
| FCs 後太短 | 酸亮但可能尖、生、甜感不足 |
| FCs 後適中 | 酸質比較圓，甜感出來，花果還在 |
| FCs 後太長 | 酸下降，甜感可能變厚，但花果容易鈍，苦味與烘焙味上升 |

第二鍋 FCs 後 `1:05` 對淺焙目標可以先接受。下一鍋主要不是拉長 FCs 後時間，而是讓 FCs 本身晚一點。

### Drop BT

Drop BT 是下豆那一刻 Artisan 讀到的豆溫。它有用，但不要單獨看。

同一台機器上可以比較：

- Drop BT 越高，通常烘焙程度越深，酸會下降，甜感/焦糖/烘焙味會增加。
- Drop BT 太低，可能保留花果，但也可能有生青、尖酸、甜感不足。
- 不同機器或不同批次量，BT 絕對值不能直接互相比。

這鍋 Drop BT `175.2°C`，搭配剛烘完有酸甜且沒有草青生味，可以先視為這台 Kaleido M1 上的淺焙可用落點。後續是否要提高 Drop，要等第 `3-5` 天正式杯測確認，不要只因為 BT 數字看起來低就急著加深。

## 第三鍋要驗證什麼

第三鍋不要一次改太多。建議只驗證一件主軸：

```text
讓 FCs 往後移，但保留第二鍋的短發展策略。
```

建議方向：

| 項目 | 第二鍋 | 第三鍋建議 |
| --- | ---: | ---: |
| 入豆 BT | `149.9°C` | 先維持 `148-150°C` |
| 起始 Burner | `50%` | 小降到 `45%` |
| DE 目標 | `3:45` | `4:00-4:20` |
| FCs 目標 | `6:24` | 先拉到 `7:00-7:45` |
| FCs -> Drop | `1:05` | 維持 `1:05-1:15` |
| Drop 判斷 | `175.2°C` / `+6.3°C` | 先抓 FCs 後 `+6-8°C`，再搭配氣味與時間 |

杯測驗證重點：

1. 酸甜是否更融合，而不是只有酸甜分開存在。
2. 苦味是否主要來自萃取，而不是烘焙尾韻。
3. 花香、檸檬、桃子、紅蘋果是否比第二鍋更清楚。
4. 第 `3-5` 天是否仍維持「沒有青草、穀物、生味」；如果仍乾淨，先不要增加發展或提高 Drop。
